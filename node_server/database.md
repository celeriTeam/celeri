### steps_competition:

```sql
CREATE TABLE steps_competition (
  id INTEGER GENERATED BY DEFAULT AS IDENTITY UNIQUE PRIMARY KEY,
  user_id TEXT NOT NULL UNIQUE,
  created_at TIMESTAMP DEFAULT NOW(),
  steps INTEGER DEFAULT 0
);
```

# SQL QUERIES

### steps_competition

- `POST` `/add-user`

    Add new user to the competition, with 0 steps.
    
    Request body:

    ```
    { 
        "user_id": string 
    }
    ```
    
    Response:
    - `200` `OK`: User successfully added.
    - `400` `Bad Request`: User already exists or validation error.

- `POST` `/update-steps`
    
    Request body:

    ```
    { 
        "user_id": string,
        "steps": int
    }
    ```
    Response:
    - `200` `OK`: Steps updated successfully.
    - `400` `Bad Request`: User not found or invalid input.
    - `500` `Internal Server Error`: Server error.

- `GET` `/data`
    
    Response:
    ```
    [
        { 
            "user_id": string,
            "steps": int,
            "created_at": timestamp
        }
    ]
    ```

- `GET` `/user-info?user_id=abc`
    
    Query:
    ```
    {
        "user_id": string
    }
    ```

    Response:
    ```
    [
        { 
            "user_id": string,
            "steps": int,
            "rank": int
        }
    ]
    ```

    Response Code:
    - `200` `OK`
    - `404` `Not Found` – User does not exist.
